# A03-4 字串(內容特徵判斷)


### (1) 常用的字串內容特徵判斷方法
```
isdigit()
isalnum()
isalpha()
isupper()
islower()
isspace()
```

### (2) 程式範例
``` python
#----------------------------------------
# 輸入字串, 判斷內容是否均為數字.
# (只有數字, 不包括小數點及負數)
#----------------------------------------

# 輸入容字串
s = input('請輸入字串: ')

# 印出原始字串內容
print('內容:', s)

# 判斷內容是否均為數字
print('是否內容為數字:', s.isdigit())
```

### (3) 程式範例
``` python
#----------------------------------------
# 輸入字串, 判斷內容是否均為英文及數字.
# (英文包括大小寫, 不包括小數點及負數)
#----------------------------------------

# 輸入容字串
s = input('請輸入字串: ')

print('內容:', s)

# 判斷內容是否均為英文及數字

```
