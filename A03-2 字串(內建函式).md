# A03-2 字串(內建函式)


### (1) 常用來處理字串的內建函式
```
chr()
ord()
len()
```

### (2) 程式範例
``` python
#------------------------------------------
# 輸入字串, 印出字串長度及第1個字的Unicode 
#------------------------------------------

# 輸入英文內容字串
s = input('請輸入字串: ')

# 印出原始字串內容, 長度及第1個字的Unicode 
print('內容:', s)
print('字串長度: ', len(s))
print('第1個字的Unicode: ', ord(s[0]))
```
