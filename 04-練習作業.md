
## 1. BMI評估

#### 程式編號: p0401 <p/>
```
由使用者輸入身高(公分)及體重(公斤), 請印出BMI值及其評估。

*註: (1) bmi值 = 體重(公斤) / 身高(公尺)的平方
         例如: 某人體重45公斤, 身高159公分, 其bmi = 45 / (1.59*1.59) = 17.8(大約值)
           
    (2) bmi的評估: 
        衛生福利部公布的成人標準:
           
        BMI < 18.5 為"過輕",
        18.5 <= BMI < 24 為"正常體重",
        24 <= BMI < 27 為"過重",
        BMI >= 27 即為"肥胖"
```

## 3. 薪水計算

#### 程式編號: p0403 <p/>
```
有一個工讀以時薪計算, 每小時150元.

如果工讀生每月出勤50小時(含)以上, 獎金1200元;
每月出勤35小時(含)以上至49小時, 獎金900元;
每月出勤20小時(含)以上至34小時, 獎金600元;
其餘沒有獎金.

工讀生每月的薪水包括時薪及獎金.

由使用者輸入工讀時數, 請印出薪水.

*註: (1) 輸入值為整數
     (2) 輸出為整數
```
