## 計算BMI

由使用者輸入的身高及體重計算BMI.

#### 1. 文件
```
輸入:
    height (float): 身高, 以公分為單位.
    weight (float): 體重, 以公斤為單位.

輸出:
    float: BMI(身體質量指數), 小數留2位餘四捨五入.

處理摘要:
    (1) 使用者輸入身高及體重.
    (2) 檢查使用者輸入的身高及體重是否為數字?
    (3) 檢查使用者輸入的身高及體重是否均為大於0?
    (4) 計算BMI=體重/身高的平方. 
        *體重以公斤為單位, 身高以公尺為單位.
    (5) BMI的小數留2位, 餘四捨五入.
    (6) 印出BMI

範例:
    >>> heiht=160
    >>> weight=45
    17.58
```

#### 2. 程式
``` python
"""
程式名稱: 計算BMI
作者: XXX
"""

# 使用者輸入身高及體重
height = input('請輸入身高(公分): ')
weight = input('請輸入體重(公斤): ')

# 將輸入資料轉成浮點數
try:
    height = float(height)
    weight = float(weight)
except:
    print('輸入有誤, 請輸入數值資料!')
    exit()

# 檢查輸入資料是否大於0
if height<=0 or weight<=0:
    print('輸入有誤, 請輸入大於0的數值資料!')
    exit()

# 計算BMI
bmi = weight / ((height/100)**2)

# BMI取小數2位, 四捨五入
bmi = round(bmi, 2)

# 印出BMI
print('BMI=', bmi)
```
