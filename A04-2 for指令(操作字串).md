# A04-2 for指令(操作字串)


### (1) for指令的格式
``` python
for指令用來迭代一個可迭代物件, 這些可迭代物件包括list, set, tuple, dict及str等.

如:
for 變數 in 序列物件:
    敘述1
    敘述2
    敘述3
    ...
```

### (2) 程式範例
``` python
#----------------------------
# 印出一個順序的文字
#----------------------------

# 輸入一個字串
data = input('請輸入一個字串: ')

# 印出一個字串中的每個文字
for s in data:
    print(s)
```
