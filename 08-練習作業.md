請以以下的程式為基礎, 完成各個程式需要完成的圖形

## 程式範例
``` python
for i in range(10):
    print('*'*(i+1))
```

## 執行結果(10行):
```
*
**
***
****
*****
******
*******
********
*********
**********
```
