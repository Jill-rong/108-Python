# A02-1 if指令


### (1) if指令的格式

if條件式內至少有一行敘述, 它一定要縮排, 表示if條件成立時才會執行的敘述; <p/>
如果if成立後要執行多行敘述, 它要一定有相同格式的縮排並寫在if指令之後.<p/>

``` python
if 條件式:
    敘述1
    敘述2
    敘述3
```

### (2) if條件式常用符號
```
>
<
>=
<=
!=
and
or
()
```

### (3) 程式範例
``` python
#-----------------------
# 此餐廳每人用餐450元,
# 超過4人(含4人)打9折
#-----------------------

# 輸入用餐人數
p = input('輸入用餐人數: ')

# 型態轉換
try:
    p = int(p)
except:
    print('請輸入整數資料!')
    exit()

# 計算餐費
t = 450 * p

if p >= 4:
    t = int(t * 0.9)

# 印出餐費
print('餐費=', t)
```
